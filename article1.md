# 生物信息学 – 第1讲课堂笔记

## 一、什么是生物信息学？

**Bioinformatics = Biology + Data Science + Computation**  

生物信息学是在大规模生物数据（基因组学、转录组学、蛋白组学等）背景下，利用计算方法对生命系统进行分析、建模和解释。  

**核心问题：**

1. 数据如何存储？  
2. 数据如何处理？  
3. 如何将“杂乱”的数据转化为“有意义”的信息？

---

## 二、现代生物数据特点（Big Data in Biology）

**现代生命科学特征：**

* 高通量测序（NGS）  
* 单细胞测序  
* 多组学整合（Multi-omics）  
* 大规模临床与分子数据整合  

**数据四大特征：**

| 特征 | 说明 |
| ---- | ---- |
| Volume | 数据量巨大 |
| Variety | 数据类型多样 |
| Velocity | 数据更新快 |
| Noise | 数据噪声复杂 |

**结论：**  
> 生物科学正从“假设驱动”逐步走向“数据驱动”。

---

## 三、Algorithm 与 Model 的区别

### 1️⃣ Algorithm

**定义：**  
一套明确、有限、可执行的步骤，用于解决具体问题。  

**特点：**

* 有输入与输出  
* 关注“如何计算”  
* 强调执行流程  
* 可以直接编程实现  

**例子：**

* 动态规划（序列比对）  
* K-means 聚类  
* 排序算法  
* 最短路径算法  

**总而言之：**  
> Algorithm 是“解决问题的方法步骤”。

---

### 2️⃣ Model

**定义：**  
对现实世界的一种抽象表达，用数学结构刻画系统规律。  

**特点：**

* 关注“系统是什么”  
* 包含假设  
* 可解释或可预测  
* 通常含有参数  

**例子：**

* 线性回归模型  
* 逻辑回归模型  
* 隐马尔可夫模型（HMM）  
* 神经网络模型  

**总而言之：**  
> Model 是“对现实的抽象假设”。

---

### 3️⃣ 二者关系对比

| 维度        | Algorithm | Model |
| --------- | -------- | ----- |
| 本质        | 计算步骤 | 抽象表达 |
| 关注点      | 怎么算   | 怎么描述 |
| 是否含假设  | 不一定   | 必须包含 |
| 举例        | 最小二乘法 | 线性回归 |

**核心理解：**  

* Model 决定结构  
* Algorithm 决定求解方式  

**例子说明：**  

* **线性回归** 是模型 → 描述输入与输出之间的关系  
* **最小二乘法** 是算法 → 求解该模型的参数

## 四、高维数据（High Dimension）

**特点：**

* 一个样本可能包含上万个基因表达值  
* 特征数远大于样本数  

**带来的问题：**

* 维度灾难（Curse of dimensionality）  
* 容易过拟合  
* 距离度量失效（高维空间中“距离”意义减弱）  

**解决思路：**

* 降维（PCA）  
* 正则化（L1 / L2）  
* 特征选择

## 五、本学期生物信息学学习计划

### 总目标

* 理解核心模型的数学本质  
* 掌握基础算法思想  
* 能够分析基础生物数据  

### 第一阶段（1–4周）：基础能力建设

* 复习线性代数、概率论与数理统计等前置知识  
* Python、GitHub等内容学习，夯实技术地基  

### 第二阶段（5–12周）：核心方法学习

* 对NGS数据分析的后续学习，力争更深入地理解课程
* 与此同时，进一步提升编程能力与数据分析能力

**实践目标：**

* 能够进行相对有效的编程
* 分析小规模真实数据

### 第三阶段（13–16周）：综合提升

* 学习神经网络基础类似的AI+内容，争取开拓思维的同时能够受到启发，有所实践

**目标：**

* 从“会使用工具”提升到“理解原理”  
* 从“做计算”提升到“做解释”
